# yui-chan
telegram-chat-bot

A cool Telegram chat bot.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Usage
- create a file with name ".config" and inside the file create:
```
TELEGRAM_TOKEN = "your token"
OPENAI_KEY = "your token"

```
## and then run Docker file

# thats' all! feel free to use! ^_^
